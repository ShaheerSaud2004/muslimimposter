# Creative brief: “Have Khafī” slide

**Target:** Muslim youth (middle school → college)  
**Tone:** Clean, modern, minimal. Warm, community, brotherhood, spiritual motivation.  
**Avoid:** Corporate, cheesy, generic AI, stock-photo feel.

---

## Layout instructions (design system)

- **Canvas:** 1080×1080 px (Instagram square).
- **Grid:** 8pt base unit. Use 8pt / 16pt / 24pt / 32pt / 40pt / 48pt for spacing.
- **Hierarchy:**
  - 1 clear headline (“Have Khafī.”).
  - 1 supporting line (Arabic “خفي” or short tagline).
  - Phone mockup as hero (screenshot inside).
  - One short line under phone (e.g. “The Islamic hidden word game — on you.”).
- **Breathing room:** Min 24pt (3×8) between headline and phone; 16pt between phone and tagline.
- **Colors:** Warm cream #FAF8F4 background; headline dark #2C2416; accent tan/gold #8B6914; secondary #6B5E4A.
- **Type:** One clean sans (e.g. system UI or Instagram Sans–style). Headline 52–56px, bold; sub 20–24px, medium. No more than 2 weights.
- **Phone frame:** Subtle shadow, rounded corners (~48px), thin dark bezel so the screenshot reads as “real phone.”

---

## How to get the slide image

1. Open `instagram/slide-have-khafi.html` in a browser (Chrome/Safari). It uses `app-screenshot.png` in the same folder.
2. Zoom so the 1080×1080 layout fills the window (or use dev tools → responsive → 1080×1080).
3. Screenshot the page or export to PNG at 1080×1080. Save as `slide-have-khafi.png` in `instagram/` for Instagram.
