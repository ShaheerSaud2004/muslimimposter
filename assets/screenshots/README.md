# App screenshots

Drop your simulator or device screenshots (PNG) here, then run:

```bash
npm run resize-app-previews        # → assets/app-previews/ (1284×2778)
npm run resize-app-previews -- --frame   # also → assets/app-previews-with-frame/ (with phone border)
```

Use the resized images for App Store Connect (iPhone 6.5" / 6.7") or Google Play. See **SCREENSHOTS_AND_PREVIEWS.md** in the project root for which screens to use on the website and in the update section.
